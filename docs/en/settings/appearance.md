# Personalization

NodeAuth supports deep visual customization, ensuring that your security tools are also an aesthetic pleasure.

---

## 🌙 Theme Mode
*   **Automatic Response**: Uses `matchMedia` to dynamically follow your operating system theme (Dark/Light).
*   **Manual Switch**: You can force a specific mode in the settings.
*   **Glassmorphism**: The interface features modern transparency and frosted glass effects, perfectly suited for high-resolution displays.

## 🌏 Language Selection
*   **Bilingual Support**: Comprehensive i18n support for both Chinese and English.
*   **Dynamic Injection**: Switching languages does not require a page refresh. All UI components (including time pickers and pagination) refresh in real-time.

## 🎚️ Layout Preferences
*   **Layout Density**:
    *   **Standard Mode**: Balanced spacing, ideal for mobile touch operations.
    *   **Compact Mode**: Reduced card padding and font sizes, suitable for displaying more accounts on a single desktop screen.
*   **Icon Customization**: Support for searching and assigning exclusive icons to your accounts from a library of over 10,000 Iconify icons.
