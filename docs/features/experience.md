# ⚡ 极致体验：顺滑交互与 PWA

一个好的安全工具不应该是冰冷的，它应该是温暖且无感的。NodeAuth 通过“离线优先”和“全生态搬家”技术，为您提供原生级别的流畅体验。

---

## 📱 1. 离线优先架构与秒开体验 (Offline-First)
NodeAuth 不只是一个“网页版”工具，它采用了先进的 PWA 技术，确保在极端环境下依然稳健：
*   **秒开体验**：应用核心逻辑已预载至本地 Service Worker，即使没有网络，打开应用也能瞬间响应界面。
*   **本地计算**：TOTP 算法（包含 Steam 专用算法）完全在前端执行。这意味着即使在飞机上、地下室等无信号区域，您依然能正常获取验证码。
*   **资源预热**：在系统空闲时间，静默预热后续可能用到的重型异步组件，保持界面零延迟。

---

## 📦 2. 全生态无缝搬家 (Smooth Migration)
从其他 2FA 工具迁移到 NodeAuth 是极为简单且无痛的。我们的智能探测引擎支持 20+ 主流 App：
*   **独立 App 类**：支持 2FAS (.2fas)、Aegis (.json)、Raivo、Ente Auth、Authenticator Pro 等导出。
*   **密码管理器类**：支持 Bitwarden (加密解析)、1Password (CSV)、KeepassXC 等。
*   **大厂协议类**：支持 Google Authenticator (分页扫码) 及 Microsoft Authenticator。
*   **离线处理**：所有迁移逻辑均在浏览器本地执行，您的所有 Secret 绝不上传服务器。

---

## 💎 3. 原生级 PWA 体验 (Native Feel)
*   **跨端安装**：支持将 NodeAuth 安装到 iOS、Android、Windows、Mac 或 Linux。
*   **独立运行**：摆脱浏览器地址栏与标签页的束缚，获得沉浸式的原生 App 视觉。
*   **Moblie Hub 中心**：移动端自动切换为底部导航模式，提供专属的枢纽站，方便单手操作所有核心功能。

---

## ✨ 4. 极致视觉与交互细节 (UI/UX)
*   **数字分段展示**：验证码采用 `3+3` 或 `2+3` 风格留白分段，极大提升眼力识别与手工录入效率。
*   **Next Code 预读**：点击倒计时环，提前滑出下一个周期的验证码，解决关键时刻的“最后 3 秒”冲突。
*   **动态动画**：层级进深感导航，模拟原生系统的层叠物理效果，让每一次点击都有回馈感。
