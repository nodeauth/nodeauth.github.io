# 项目介绍与预览

NodeAuth 是一款**高安全**、**轻量级**、**跨平台**的二步验证 (2FA/TOTP) 管理工具。

无论是追求极致性价比的 **Serverless 部署**，还是追求数据绝对控制权的**原生私有化部署**，NodeAuth 都能完美适配，为您提供如原生 App 般丝滑的 2FA 密钥管理体验。

---

## 💎 设计理念：双位一体的极致追求

NodeAuth 不仅仅是一个工具，它是我们对**极致安全**与**极致体验**不懈追求的产物。

### 🛡️ 极致安全 (Ultimate Security)
**“不可逾越的数据基石”**
*   **全流程加密**：核心数据采用 **WebAuthn 硬件盐** 注入及 AES-GCM 高强度冷加密。
*   **主权归属**：无论是 D1 还是 MySQL，数据始终锁在您指定的存储中。
*   **主动防线**：内置环境嗅探、防偷窥模糊及自愈同步机制，全天候守护您的身份资产。
> [➔ 探索安全防线细节](./security)

### ⚡ 极致体验 (Ultimate Experience)
**“无感顺滑的交互艺术”**
*   **原生级 PWA**：一键安装至桌面或主屏，支持完全离线运行，动态码从不“断电”。
*   **无缝搬家**：支持 Bitwarden、Google Auth 等 20+ 主流 App 数据的一站式迁移。
*   **微动效设计**：数字分段展示、Next Code 预读、平滑过渡动画，让每一次解锁都从容优雅。
> [➔ 探索体验交互细节](./experience)

## 💡 为什么选择 NodeAuth？

我们相信安全不应以牺牲便捷为代价。NodeAuth 通过先进的 **WASM 核心** 和 **离线优先架构**，在保障隐私的极端环境下，依然提供了媲美顶级商业软件的使用体验。
