# PWA 安装指南

NodeAuth 是一个**渐进式 Web 应用 (PWA)**。安装后，它可以像原生 App 一样出现在您的手机桌面或电脑任务栏中，并获得更好的全屏体验和离线能力。

---

## 📱 在移动端安装 (iOS / Android)

### 1. iOS (iPhone / iPad)
由于苹果系统的限制，请务必使用 **Safari 浏览器**打开：
1. 点击底部任务栏中间的“**分享**”图标 (一个带上箭头的方框)。
2. 在弹出的菜单中向下滚动，点击“**添加到主屏幕**” (Add to Home Screen)。
3. 点击右上角的“添加”，NodeAuth 就会出现在您的桌面上。

### 2. Android (安卓)
建议使用 **Chrome 浏览器**或自带浏览器：
1. 点击浏览器右上角的“三个点”菜单。
2. 找到并点击“**安装应用**”或“**添加到主屏幕**”。
3. 确认后，应用图标将出现在您的桌面或应用抽屉中。

---

## 💻 在桌面端安装 (Windows / macOS / Linux)

建议使用 **Chrome** 或 **Edge** 浏览器：
1. 在地址栏右侧，您会看到一个类似于“**安装**”的小图标 (笔记本带箭头的图标)。
2. 点击该图标，选择“**安装**”。
3. NodeAuth 现已作为独立窗口运行。您可以将其固定到任务栏或 Dock。

---

## ✨ 安装后的优势
*   **沉浸体验**：隐藏浏览器的地址栏、工具栏，将每一寸屏幕空间留给数据。
*   **离线秒开**：通过本地缓存实现“白屏零感知”启动。
