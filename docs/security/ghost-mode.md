# 防窥模式

防窥模式旨在保护您在物理社交场景下的数据隐私，防止侧目窥视或屏幕预览泄密。

---

## 👻 核心功能

### 1. 自动高斯模糊
开启后，NodeAuth 会实时监听浏览器的 `visibilitychange` 事件：
*   **切后台瞬间生效**：当您在手机上上划切换应用，或者在电脑上切换窗口时，NodeAuth 界面会瞬间覆盖一层 **15px 的高斯模糊**。
*   **防止任务快照**：这能有效防止操作系统在“任务管理/预览”界面生成包含您 2FA 动态码的缩略图。

### 2. 隐私长按查看
默认状态下，列表中的验证码会以 `••••` 的形式锁定：
*   **长按显示**：您必须用手指长按该条目 250ms 以上，真实代码才会淡入显示。
*   **防误触**：抬手即恢复，极大降低了在人群密集的地铁或电梯中代码泄露的概率。

### 3. 操作脱敏提示
在涉及导入、导出或重置主密钥等高危操作时，系统会自动调低背景透明度并移除显眼的红点提示。

## 🛠️ 如何配置
1. 前往“系统设置” -> “**防窥设置**”。
2. 分别开启“防偷窥模式 (后台模糊)”与“隐私列表展示”。
