# 安全锁定

安全锁定是 NodeAuth 的“应用级门禁”，防止您的手机在未锁定状态下被他人偷窥数据。

---

## 🔢 6位 PIN 码
这是最通用的锁定方式：
1. 在“安全中心” -> “**安全锁定**”中开启。
2. 设置一个 6 位数字密码。
3. **哈希策略**：系统在本地通过 **100,000 次 PBKDF2** 迭代进行哈希处理，物理防御暴力破解。
4. **震动反馈**：如果输入错误，手机会通过 0.4s 的 Shake 动画和物理震动提醒您。

## 👆 生物特征解锁 (本地)
如果您的设备支持指纹、面容或屏下指纹：
*   开启 PIN 码后，可以勾选“**生物识别快捷进入**”。
*   **眨眼即开**：进入应用 300ms 后会自动拉起系统指纹识别，无需手动输入数字。

## ⏲️ 自动锁定延迟
您可以自定义锁定的时机：
*   **立即**：切换浏览器标签或切后台瞬间即锁定。
*   **闲置锁定**：例如设定为“5分钟后锁定”。

## ☢️ 核平自毁
如果您遗忘了 PIN 码，在锁屏界面有连点 5 次触发**“核平地注销”**的隐藏选项。
*   **后果**：会物理抹除该设备的所有本地缓存数据，包括 `enc_device_salt`（核心加密盐）、`bio_cred_id`（生物识别凭证）及所有 `LocalStorage` 记录。您需要重新登录，并联网重新拉取数据。
