# 设备与会话管理

NodeAuth 提供透明的会话审计功能，让您随时掌控谁在什么时间、什么设备上登录了您的系统。

---

## 📱 实时会话审计
在“安全中心” -> “**设备与会话**”中，您可以查看到当前所有处于登录态的记录：
*   **平台识别**：自动映射超过 10 种品牌图标（如 MacOS、Windows、iOS、Android 等）。
*   **位置与 IP**：显示登录时的公网 IP。
*   **隐私脱敏**：系统会对 IP 地址执行**子网掩码动态遮蔽**，在提示所属区域的同时保护精确物理位置。

## 🚫 远程安全注销
如果您发现有异常登录，或者在他人电脑上登录后忘记退出：
1. 找到对应的设备条目。
2. 点击右侧红色的“**强制注销**”图标。
3. 系统将物理删除后端会话令牌，前端设备在 5 秒内将自动跳回登录页并清空缓存。

## 🔐 唯一设备盐 (Device Salt)
*   每个设备在绑定 Passkey 或登录后，系统会注入一个唯一的 `sys:sec:device_salt` 到本地 IDB。
*   这保证了即便在不同设备登录同一个账号，各端的解密环境也是硬件级隔离的。
