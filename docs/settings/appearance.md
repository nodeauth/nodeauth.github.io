# 个性化设置

NodeAuth 支持深度的视觉自定义，让工具也具备审美享受。

---

## 🌙 主题模式
*   **自动响应**：利用 `matchMedia` 动态监听您的操作系统主题（深色/浅色）。
*   **手动切换**：您可以在设置中强制指定一种模式。
*   **玻璃拟态**：界面采用了现代的透明度与毛玻璃效果，适配各种高分辨率显示器。

## 🌏 语言切换
*   **中英双语**：完善的 i18n 指引。
*   **动态注入**：切换语言无需刷新页面，全量 UI 组件（包括时间选择器、分页符）会实时刷新。

## 🎚️ 界面布局偏好
*   **布局密度**：
    *   **标准模式**：间距适中，适合移动端指尖操作。
    *   **紧凑模式**：缩小卡片 Padding 和字体，适合在电脑上单屏展示更多账号。
*   **图标美化**：支持从超过 10,000 个 Iconify 图标库中搜索并为您的账号分配专属图标。
