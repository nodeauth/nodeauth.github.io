# 2FA APP 生态排行榜

在选择 2FA 工具时，安全性、开放度和数据主权是三大衡量标准。以下是 NodeAuth 为您整理的行业横向对比参考。

---

## 🚩 重点避坑名单
*   **Authy**：虽然用户量大，但其**拒绝提供任何导出功能**。一旦使用，您的账号将被永久锁定在其生态内。NodeAuth 强烈建议您将其迁移出来。
*   **Microsoft Authenticator**：虽支持云同步，但其备份机制在大厂账号异常时可能不可靠。

## 🏆 推荐开源生态
| App 名称 | 开源度 | 数据导出 | 离线能力 | 评价 |
| :--- | :--- | :--- | :--- | :--- |
| **NodeAuth** | ✅ 全开源 | ✅ 极佳 | ✅ 强缓存 | 跨端、0成本、数据自持 |
| **2FAS** | ✅ 开源 | ✅ 友好 | ✅ 离线 | 移动端交互体验极佳 |
| **Aegis** | ✅ 开源 | ✅ 友好 | ✅ 离线 | 安卓端最强的开源选择 |
| **Ente Auth** | ✅ 开源 | ✅ 友好 | ✅ 离线 | 多端同步做得较好 |

## 📊 评估指标说明
1. **开源度**：代码是否公开可查，是否存在后门。
2. **导出自由度**：是否允许用户以 JSON 或 CSV 这种通用格式拿走自己的数据。
3. **离线能力**：在没有网络的情况下，算法是否能独立运行。
4. **备份多样性**：是否支持用户自己掌控备份文件（如存入 NAS 或 S3）。

> [!TIP]
> **迁移建议**：如果您还在使用“导出锁死”的 App，建议趁着旧设备还在手，尽快手动或通过扫码搬家到 NodeAuth。
