# 辅助小工具

除了核心的 TOTP 功能，NodeAuth 还内置了一些提升极客使用效率的辅助工具。

---

## 🔐 密码生成器
如果您在注册网站时需要一个高强度的密码：
*   **特性**：支持自定义长度（最高 128 位）、包含特殊字符、数字及大小写。
*   **离线生成**：采用 `crypto.getRandomValues` 熵源，保密性极高。
*   **一键复制**：点击生成的密码即可自动复制到剪贴板。

## 🔍 二维码解析器
处理非标准的 2FA 图片或截图：
*   **本地解析**：直接拖入一张包含二维码的图片。
*   **协议识别**：自动解析其中的 `otpauth://` 文本并跳转到添加账号界面。
*   **隐私保护**：图片在本地内存中解析，不上传任何云端。

## ⏰ 高精度时间校准
*   **时差对齐**：系统会自动抓取服务器响应头带回的时间戳，并计算本地误差。
*   **自动补偿**：在计算验证码时，系统会自动应用该误差，无需用户手动干预。
