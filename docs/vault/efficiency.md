# 账号管理 (排序、分类与搜索)

当您的账号数量达到几十个甚至上百个时，高效的管理工具就显得尤为重要。

---

## 📁 账号分类
您可以为由于不同的用途而对账号进行归类（如：工作、社交、金融）：
1. 在侧边栏或 Tab 页切换至“分类管理”。
2. 创建一个新图标及其分类名称。
3. 在编辑账号时，选择对应的分类。
4. 在主列表顶部点击分类标签，即可一键筛选出该类目下的所有账号。

## 🔍 实时搜索
*   **关键词匹配**：主列表上方的搜索栏支持对“账号名”、“备注”及“发行方 (Issuer)”进行实时模糊搜索。
*   **快捷清空**：切换 Tab 或点击清空按钮，搜索状态会自动复位。

## 📶 自定义排序
NodeAuth 提供多种排序方式，满足不同的使用习惯：

### 1. 拖拽排序
*   开启排序模式（或直接长按卡片），您可以手动拖拽卡片调整其前后顺序。
*   排序结果会实时持久化到云端，保证多端一致。

### 2. 策略排序
在设置页面，您可以选择预设排序逻辑：
*   **首字母排序 (A-Z)**：按账号名称字母顺序排列。
*   **最近使用**：将最常点击复制的账号排在最前方。
*   **添加时间**：按添加的先后顺序排列。

## 🎚️ 界面密度调整
在“外观设置”中，您可以切换：
*   **卡片网格 (Card Mode)**：默认模式，展示图标，视觉更清晰。
*   **紧凑列表 (List Mode)**：单行展示，屏幕利用率高，适合账号极多的用户。
